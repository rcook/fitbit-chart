# fitbit-demo

Demonstrate Fitbit web API calls using Haskell via Req

## Clone repository

```
git clone https://github.com/rcook/fitbit-demo.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build --fast
```

## Run application

```
stack exec fitbit-demo-app
```

## Run tests

```
stack test
```
