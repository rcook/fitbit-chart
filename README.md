# fitbit-chart

Generate Flot charts from Fitbit data in AWS Lambda

## Clone repository

```
git clone https://github.com/rcook/fitbit-chart.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build --fast
```

## Run application

```
stack exec fitbit-chart-cli
```

or

```
stack exec fitbit-chart-lambda
```
